# Life Targets - User Guide

## Overview
Life Targets is a comprehensive goal-planning feature that helps you set, analyze, and track quarterly goals across four key life domains: **Body**, **Being**, **Balance**, and **Business**.

## Getting Started

### 1. Access Life Targets
- Navigate to **Life Targets** from the sidebar menu
- You'll see the main dashboard with your current quarter's progress

### 2. Set Quarterly Targets
1. Click **"Set Quarterly Targets"**
2. Select your quarter (Q1, Q2, Q3, or Q4)
3. For each domain, enter:
   - **Target**: What you want to achieve (required)
   - **Context**: Why this matters to you (optional)

**Example Targets:**
- **Body**: "Run 5k in under 25 minutes"
- **Being**: "Meditate for 20 minutes daily"
- **Balance**: "Have 12 family dinners this quarter"
- **Business**: "Launch new product feature to 100 beta users"

### 3. Analyze Measurability
After setting targets, click **"Analyze"** to:
- Get clarity scores (0-10) for each target
- See AI-generated suggestions for more specific, measurable targets
- Apply suggestions with one click

### 4. Generate Monthly Missions
Break down your quarterly targets into 3 monthly missions:
1. Navigate to **"Generate Monthly Missions"**
2. AI generates specific milestones for months 1, 2, and 3
3. Each mission includes:
   - **Mission**: What to accomplish
   - **Why**: Rationale for this step

### 5. Set Daily Actions
Get actionable daily habits to achieve your targets:
1. Visit **"Get Daily Actions"**
2. Review 10 suggested actions per domain
3. Select your favorite and set it as your **Daily Habit**

## Features

### Quarter Switching
- Change quarters using the selector in the form header
- Each quarter has independent targets and tracking

### Progress Tracking
The dashboard shows:
- Number of targets set (out of 4)
- Whether missions are generated
- Number of daily habits established
- Overall completion status

### Auto-Save
- Changes save automatically as you type
- Visual indicator confirms successful saves

### Keyboard Shortcuts
- **Tab**: Navigate between fields
- **Ctrl/Cmd + S**: Save form (when enabled)
- **Escape**: Close modals

## Troubleshooting

### "Analyze" button disabled?
- Ensure at least one target is set
- Targets must have text content

### Analysis not appearing?
- Check your internet connection
- Wait 5-10 seconds for AI processing
- Try refreshing the page

### Can't save changes?
- Verify you're logged in
- Check for error messages
- Contact support if issues persist

## Best Practices
1. **Set targets at quarter start**: Review and update at the beginning of each quarter
2. **Review weekly**: Check progress on missions and habits
3. **Iterate**: Refine targets based on clarity scores
4. **Be realistic**: Set challenging but achievable goals
5. **Use all domains**: Balance across all four life areas
